# Müşteri Memnuniyetini Tahmin Etme Projesi

Bu proje, bir restoranın müşteri verilerini kullanarak, müşterilerin memnuniyet durumlarını tahmin etmeyi amaçlayan bir veri bilimi çalışmasıdır.

## Proje İçeriği
- **data/**: Projede kullanılan ham ve işlenmiş veri setlerini içerir.
- **notebooks/**: Veri analizi (EDA), temizleme ve modelleme süreçlerini içeren Jupyter not defterleri yer alır.
- **Müşteri Memnuniyetini Tahmin Etme.pdf**: Projenin detaylı raporudur.

## Kullanılan Teknolojiler
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn

## Önemli Bulgular
- Modelimiz müşteri memnuniyetini tahmin etmede %85 doğruluk oranına ulaşmıştır.
- Müşteri memnuniyetini en çok etkileyen temel faktörlerin başında "Ürün Kalitesi" gelmektedir.
